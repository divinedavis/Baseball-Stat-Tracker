# Baseball Stat Tracker

Record your kids' baseball stats.

SwiftUI iOS app for tracking per-player stats (AVG / OBP / SLG / OPS and the counting stats behind them).

## Build

```bash
xcodegen generate
open BaseballStatTracker.xcodeproj
```

Deployment target iOS 17. No external packages.
